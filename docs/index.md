# Getting started

## Installation

### Prerequisites

- [Node.js](https://nodejs.org/) version 20 or higher.

::: code-group

```bash [npm]
npm add -D dependency-owners
```

```bash [pnpm]
pnpm add -D dependency-owners
```

```bash [yarn]
yarn add -D dependency-owners
```

:::
